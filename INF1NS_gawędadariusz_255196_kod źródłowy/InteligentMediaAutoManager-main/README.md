# InteligentMediaAutoManager using:

- Python 3.8.5
- Django 3.2
- PyCharm IDE
- HTML5 / CSS
- JQuery 3.6.0 + JQuery UI
- MongoDb

##commands need to run code 
` pip install selenium`
` pip install instaPy`
` pip install webdriver-manager`
` pip install pymongo`
